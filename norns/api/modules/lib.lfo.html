LFO:add ([shape[, min[, max[, depth[, mode[, period[, action]]]]]]])
construct an LFO via table arguments eg. my_lfo:add{shape = 'sine', min = 200, max = 12000}

Parameters:

  • shape string The shape for this LFO (options: 'sine','saw','square','random'; default: 'sine') (optional)
  • min number The minimum bound for this LFO (default: 0) (optional)
  • max number The maximum bound for this LFO (default: 1) (optional)
  • depth number The depth of modulation between min/max (range: 0.0 to 1.0; default: 0.0) (optional)
  • mode string How to advance the LFO (options: 'clocked', 'free'; default: 'clocked') (optional)
  • period number The timing of this LFO's advancement. If mode is 'clocked', argument is in beats. If mode is 'free', argument is in seconds. (optional)
  • action function A callback function to perform as the LFO advances. This library passes both the scaled and the raw value to the callback function. (optional)
